# 🌐 Portfolio — Marco Rossi

Portfolio personale costruito con **HTML, CSS e JavaScript vanilla** — nessun framework, nessuna dipendenza.

🔗 **Live:** [tuousername.github.io/portfolio](https://tuousername.github.io/portfolio)

---

## ✨ Caratteristiche

- Design responsive (mobile-first)
- Animazioni CSS al caricamento e allo scroll
- Font Google: Syne + Manrope + JetBrains Mono
- Sezioni: Hero, Chi sono, Competenze, Progetti, Contatti
- Codice commentato e facile da personalizzare

## 🛠 Tecnologie

| Tecnologia | Utilizzo |
|---|---|
| HTML5 | Struttura semantica della pagina |
| CSS3 | Layout (Flexbox/Grid), variabili CSS, animazioni |
| JavaScript | IntersectionObserver per lo scroll reveal |
| Google Fonts | Tipografia |
| GitHub Pages | Hosting gratuito |

## 📁 Struttura

```
portfolio/
└── index.html    ← tutto il codice in un unico file
```

## 🚀 Come eseguire localmente

```bash
# Clona la repo
git clone https://github.com/TUOUSERNAME/portfolio.git

# Apri il file nel browser
open index.html
# oppure trascina index.html nel browser
```
